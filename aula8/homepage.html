O que é HTML?

HTML (Linguagem de Marcação de HiperTexto) é o bloco de construção mais básico da Web. Define o significado e a estrutura do conteúdo da web. documentos HTML podem ser interpretados por navegadores. A tecnologia é fruto da junção entre os padrões HyTime e SGML. HyTime é um padrão para processamento a representação estruturada de hipermídia e conteúdo baseado em tempo. Os hipertextos são a junção de vários elementos - palavras, vídeos e conteúdos -, que, quando conectados, estabelecem uma rede de dados que permite a comunicação, o armazenamento e o compartilhamento de informações. Por exemplo, ao visitar um website, você encontra diversas informações com formatações diferentes, como parágrafos, bullets e fontes distinções. Pois estão, essa estruturação é realizada por meio do HTML.

Qual a função do HTML?

Imagine que você precisa construir um site do zero com imformações da sua empresa. Nesse caso, você precisará conhecer a estrutura do HTML, cirando um documento no bloco de notas e acrescentadno códigos - tags-, que indicação parágrafos, quebra de texto, espaçamento, imagens, entre outros. Em geral, em um único site, existem diversos documentos HTML, criando páginas como homepage, página de produto e informações sobre a empresa. Em cada aba, existe um código diferente.

CSS

Cascating Style Sheets (CSS) é usado para formatar o layout de uma página da web. Com CSS, você pode controlar a cor, a fonte, o tamanho do texto, o espaçamenteo entre os elementos, como os elementos são posicionados e dispostos, quais imagens de fundo ou cores de fundo devem ser usadas, exibições diferentes para dipositivos e tamanhos de tela diferents e muito mais!

JavaScript

JavaScript é uma linguagem de programação interpretada estruturada, de script em alto nível com tipagem dinâmica fraca e multiparadigma. Juntamente com HTML e CSS, o JavaScript é uma das principais tecnologias da World Wide Web.
